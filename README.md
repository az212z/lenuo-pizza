# وود فاير بيتزا لينو — موقع تعريفي

موقع ثابت (Static) فاخر بالعربية (RTL) لمطعم **وود فاير بيتزا لينو** في جدة — بيتزا فرن الحطب الإيطالية على طريقة نابولي ومعجنات وباستا.

## الرابط المباشر
بعد النشر على GitHub Pages:
`https://az212z.github.io/batch9-restaurants/lenuo-pizza/`

## نظرة عامة
- **التقنية:** HTML + CSS + JavaScript فانيلا بدون أي خطوة بناء (no build step) — لا `node_modules` ولا `package.json`.
- **الخطوط:** El Messiri / Reem Kufi للعناوين، Tajawal للنصوص (Google Fonts).
- **الهوية:** فحم محروق + خشب دافئ + أحمر طماطم + أخضر ريحان + برتقالي جمر — أجواء نابولي أصيلة.

## الموشن التوقيعي (Signature Motion)
هيرو **فرن الحطب**:
- قرص بيتزا SVG يدور ببطء بينما تستقر الإضافات عليه (ببيروني/ريحان/جبن تظهر بـ stagger).
- فم فرن يتوهّج بألسنة لهب متعددة الطبقات (برتقالي إلى ذهبي) وشرر يتصاعد للأعلى.
- يعمل بـ transform/opacity فقط (60fps).
- **عند `prefers-reduced-motion`:** يظهر القرص النهائي مركّبًا مع توهّج ثابت — بدون دوران أو لهب.

## النموذج (Form)
نموذج الحجز/الطلب **تجريبي للعرض فقط**:
- يتحقق من المدخلات، يبني ملخصًا، يحفظ في `localStorage` (مفتاح `lenuo_reservations`)، ويعرض إشعار نجاح (toast).
- **لا يوجد backend** — لا يُرسل بيانات لأي خادم. عند التسليم يمكن ربطه ببريد/واتساب/أداة حجز.

## التواصل
- **الهاتف (أرضي):** `0126140663` — قابل للاتصال عبر `tel:` فقط (ليس واتساب).
- **الموقع:** زر بارز لخرائط قوقل.
- **أزرار عائمة (FABs):** اتصال + خرائط فقط — **لا يوجد واتساب**.

> **ملاحظة للمالك:** لإضافة واتساب لاحقًا، زوّدنا برقم جوال دولي (مثال `9665XXXXXXXX`) ونضيف زرًا/FAB لواتساب وروابط `wa.me` في النموذج. لم نخترع أي رقم جوال.

## مصدر الصور
صور حقيقية من ملف المطعم على خرائط قوقل (`assets/img/lp-*.jpg`). تمت **تنقيتها يدويًا**: استُبعدت لوحة الأسعار/المنيو، والصور المكرّرة/الضعيفة، وصورة عصير غير مناسبة. المُستخدَم: بيتزا مارغريتا/ببيروني/خضار، باستا روبيان وبولونيز، شوربة طماطم، الواجهة، والقاعة الداخلية.
